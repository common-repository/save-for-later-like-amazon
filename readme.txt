== Description ==

Save for Later Plugin , Specifically made for WooCommerce . It's being Used by millions, Save For Later helps in enhancing the conversion rate on the website . <strong> Generating More Sales </strong>. Can add your product on a temporary cart for making them purchased at a later stage. To get started: activate the Save For Later plugin.

Major features in "Save for Later"  include:

* Helps in binding customer cart items even after an year
* Flexibility to customer to add less important items in cart
* Free to use

== Installation ==

Upload the Save For Later plugin to your WooCommerce Wordpress Store, Activate it, then under setting > Save For Later you can manage it 

1, 2, 3: You're done!
